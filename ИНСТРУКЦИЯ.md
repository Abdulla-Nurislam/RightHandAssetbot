# Almaty RE-Tech Club Bot

> Закрытое Telegram-пространство для лидеров и инвесторов рынка недвижимости и технологий. Бот выполняет роль цифрового консьержа, системы скрининга новых резидентов и инструмента для презентации эксклюзивных предложений.

## ✨ Ключевые возможности

- 🔐 **Верификация через Google Sheets:** Доступ к меню только для подтвержденных резидентов из реестра.
- 🛡 **Автоматический скрининг:** Сбор заявок на вступление от новых пользователей с автоматической фильтрацией по стоп-словам (казино, ставки и т.д.).
- 📅 **Бронирование событий:** Интерактивное меню записи на закрытые ужины, форумы и презентации.
- 💎 **Привилегии и Воронки:** Трехуровневая система эксклюзивных предложений с пошаговыми FSM-сценариями для сбора квалифицированных лидов (Недвижимость, Квартиры, Автомобили).
- 📊 **Интеграция CRM:** Все заявки и лиды мгновенно и асинхронно записываются в Google Sheets (лист «Лиды», «waitlist», «Approved»).

## 🚀 Стек технологий

- Python 3.10+
- `aiogram 3.x` — современный асинхронный фреймворк для Telegram-ботов
- `gspread` + `google-auth` — работа с Google Таблицами через Service Account

## 🛠 Запуск проекта

1. **Клонируйте репозиторий и установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройте переменные в коде:**
   В файле `bot.py` укажите свои данные:
   ```python
   BOT_TOKEN: str = "ВАШ_ТОКЕН_ОТ_BOTFATHER"
   SPREADSHEET_ID: str = "ID_ВАШЕЙ_GOOGLE_ТАБЛИЦЫ"
   ```

3. **Добавьте ключ Google Cloud:**
   Положите файл `credentials.json` (Service Account) в корень проекта.

4. **Запустите бота:**
   Обязательно используйте скрипт `start.sh` для автоматического управления процессами:
   ```bash
   ./start.sh
   ```
