## 1. 🔑 Как настроить бота «под себя» (первый запуск)

Откройте файл `bot.py` в любом текстовом редакторе и измените раздел **КОНФИГУРАЦИЯ** (строки 48–56):

### 1.1 Токен бота
```python
BOT_TOKEN: str = "ВАШ_ТОКЕН"
```
Получить токен: напишите боту [@BotFather](https://t.me/BotFather) → `/newbot` → скопируйте токен.

### 1.2 Куда приходят заявки (LEADS_CHAT_ID)

Это **самая важная настройка**. Бот отправляет карточку лида в указанный чат.

#### Вариант А: Лиды приходят в личку владельцу
1. Напишите боту [@userinfobot](https://t.me/userinfobot) — он покажет ваш числовой ID
2. Вставьте число:
```python
LEADS_CHAT_ID: int = 123456789  # ваш личный Telegram ID
```
3. **Важно:** владелец должен хотя бы раз написать `/start` вашему боту!

#### Вариант Б: Лиды приходят в закрытый канал / группу
1. Создайте канал или группу в Telegram
2. Добавьте бота в канал **как администратора** (с правом отправлять сообщения)
3. Узнайте ID канала — перешлите любое сообщение из канала боту [@getmyid_bot](https://t.me/getmyid_bot)
4. Вставьте ID (начинается с `-100`):
```python
LEADS_CHAT_ID: int = -1001234567890  # ID вашего канала/группы
```

#### Вариант В: Лиды в Google Таблицу (через Zapier / Make)
Это делается без изменения кода бота. Подключите Zapier/Make к каналу (вариант Б), и он будет автоматически записывать лиды в таблицу.

## 2. 💰 Как изменить цены и тексты

В файле `bot.py` найдите словарь `TEXTS` (строки ~70–160). Это **единственное место**, где хранятся все тексты интерфейса.

### Примеры:

| Что изменить           | Найти (Ctrl+F)    | Заменить на      |
|------------------------|--------------------|-------------------|
| Цена аренды офиса      | `399 000 ₸/мес`   | Новая цена        |
| Стоимость квартиры     | `54 990 000 тенге` | Новая стоимость   |
| Цена автомобиля        | `45 000 000 ₸`     | Новая цена        |
| Модель Maserati        | `Ghibli / Levante` | Другая модель     |
| Пробег                 | `15 000 км`        | Другой пробег     |

> ⚠️ Не удаляйте теги `<b>` и `</b>` — они делают текст жирным.

## 3. 📸 Как заменить фотографии

| Папка                      | Что внутри          |
|----------------------------|---------------------|
| `commercial real estate/`  | Фото офиса          |
| `RAMS_PHOTOS/`             | Фото квартиры       |
| `MASERATI_PHOTOS/`         | Фото автомобиля     |

### Инструкция:
1. Откройте нужную папку
2. Удалите старые фото
3. Скопируйте новые (`.jpg`, `.jpeg`, `.png`, `.webp`)
4. Перезапустите бота

### ⚠️ Важные правила:
- **Размер файла:** не более 2МБ на фото (тяжёлые файлы вызывают таймаут)
- **Количество:** максимум 5 фото отправляются в альбоме
- **Порядок:** файлы сортируются по алфавиту. Называйте `01.jpg`, `02.jpg` для контроля
- Сжать фото можно на [tinypng.com](https://tinypng.com/) или [squoosh.app](https://squoosh.app/)

## 4. 🕐 Как изменить временные слоты

В `bot.py` найдите `TIME_SLOTS` (строка ~58):
```python
TIME_SLOTS = [
    ("🌅 10:00 – 12:00", "10:00–12:00"),
    ("☀️ 12:00 – 14:00", "12:00–14:00"),
    ...
]
```
Первое значение — текст кнопки, второе — что запишется в заявку.

## 5. 🗄 База данных лидов

Все заявки дублируются в файл `leads.db` (SQLite). Даже если уведомление не ушло владельцу (проблемы с интернетом), лид сохранён в базе.

### Просмотр лидов:
```bash
sqlite3 leads.db "SELECT id, created_at, direction, phone, delivered FROM leads ORDER BY id DESC;"
```

### Проверка недоставленных лидов:
```bash
sqlite3 leads.db "SELECT * FROM leads WHERE delivered = 0;"
```

### Структура таблицы:

| Поле        | Описание                              |
|-------------|---------------------------------------|
| id          | Номер заявки                          |
| created_at  | Дата и время                          |
| direction   | Направление (Коммерция/Rams/Maserati) |
| phone       | Номер телефона клиента                |
| answers     | JSON с ответами (деятельность, оплата)|
| delivered   | 1 = доставлено, 0 = не доставлено     |

## 8. ❓ FAQ

**В: Заявки не приходят в канал.**
О: Бот должен быть добавлен в канал **как администратор**. Проверьте `LEADS_CHAT_ID`.

**В: Заявки не приходят в личку.**
О: Получатель должен написать `/start` боту хотя бы один раз.

**В: Фото не отправляются / таймаут.**
О: Сожмите фото до размера менее 2МБ через [tinypng.com](https://tinypng.com/).

**В: Бот не запускается после правок.**
О: Проверьте, не удалили ли кавычку `"` или скобку `)`. Сравните с оригиналом на GitHub.

**В: Как перенести бота на сервер?**
О: Скопируйте всю папку на VPS, установите Python 3.12+ и `pip install aiogram`, запустите `python3 bot.py`. Для фоновой работы используйте `systemd` или `screen`.
